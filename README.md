# Th3-Bot

Th3-Bot is a powerful multi-purpose Discord bot designed for the Th3 community. It combines moderation, automation, utility tools, entertainment features, and server management into a single easy-to-use bot.

## Features

### 🛡️ Moderation & Security
- Advanced moderation commands
- Anti-nuke protection system
- Auto-moderation tools
- Anti-spam protection
- Anti-invite detection
- Anti-link filtering
- Anti-caps protection
- Custom word and content filters

### ⚙️ Automation
- Automatic role assignment (Autorole)
- Custom auto-reactions
- Guild-specific triggers and responses
- Server management utilities

### 🎮 Games & Fun
- Chess
- Blackjack
- Connect Four
- Tic-Tac-Toe
- Who's That Pokemon (Custom Built Game with its own leaderboard Local/Global)
- Additional mini-games and interactive commands

### 🧰 Utilities
- AI-powered helper commands
- Image generation and image tools
- Calculators and converters
- Leveling system
- Giveaways
- Various server utility commands
